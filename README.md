# :gem: Crystal Blog React Application

## :pencil2: About The Project

## Installation and usage
 
**Clone repository:**
 
 ```
https://github.com/vladimirmitrev/crystals-blog.git
 ```
 
 * To run server app
    <br/>
 ```
    cd server
    node .\server.js
 ```
 * To run client app
 ```
    cd client
    npm install
    npm run dev
 ```
 
 <br/>
